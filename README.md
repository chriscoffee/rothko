## Introduction

The CSS Filters experiment that Mark Rothko could only dream of. This normally lives at [rothko.me](http://www.rothko.me).

## Setup

```
bundle install
```

## Running

```
bundle exec jekyll serve
```

Visit http://127.0.0.1:4000 and you should see your page.
